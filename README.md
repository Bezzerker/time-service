# Time Service

## О проекте

Это приложение, которое предоставляет REST API для получения текущего времени сервера с точностью до секунд вместе с часовым поясом.

## Выбор Web MVC вместо WebFlux

В этом проекте был сделан выбор в пользу Web MVC вместо WebFlux. Это решение обосновано тем, что работа со временем является нересурсоемкой операцией, которая не приводит к простоям.

WebFlux - это реактивная система, которая лучше всего подходит для обработки большого количества одновременных запросов, где основная нагрузка связана с вводом-выводом, например, при работе с базами данных или сетевыми ресурсами. В нашем случае, однако, основная нагрузка связана с вычислением текущего времени, которое является относительно простой и быстрой операцией.

Кроме того, Web MVC предоставляет более традиционный подход к разработке веб-приложений, который может быть более понятен и предсказуем для многих разработчиков. Это может упростить обслуживание и поддержку приложения в будущем.

## Задействованные технологии

- Spring Boot 3.1
- Java 21
- Gradle 8.7

## Тестирование

Корректность реализации подтверждена с помощью модульных и интеграционных тестов на JUnit5. Тесты написаны для проверки корректности работы конечной точки API и формата ответа.

## Установка и запуск

1. Склонируйте репозиторий:
```shell
git clone https://github.com/Bezzerker/time-service.git
```
2. Перейдите в каталог проекта:
```shell
cd time-service
```
3. Соберите проект с помощью Gradle:
```shell
./gradlew clean bootJar
```
4. Запустите приложение:
```shell
java -jar ./build/libs/time-service.jar
```
